# Claims Ledger

Единый список публичных фактов для текстов РемСД. Автор текстов берет формулировки отсюда, ревьюер проверяет, что новые тексты не добавляют неподтвержденные обещания.

## Needs Proof

| Сущность | Утверждение | Что нужно подтвердить |
|---|---|---|
| КАМАЗ | видимый сертификат или документ официального статуса | документ, фото или реквизит, который можно показать на сайте |
| Ремонтная база | конкретное оборудование, станки, склад, количество постов | фото, список оборудования или подтверждение владельца |
| Процесс ремонта | фиксированная стоимость до начала работ | регламент или подтверждение владельца |
| Процесс ремонта | подготовка запчастей до прибытия техники | подтвержденный рабочий процесс |

## Blocked

| Тип утверждения | Правило |
|---|---|
| Сроки ремонта | Не писать без подтвержденного регламента по конкретной услуге. |
| Цены и скидки | Не писать без утвержденного прайса. |
| Гарантии | Не писать без подтвержденных гарантийных условий. |
| Рейтинги, отзывы, проценты, "лучший" | Не писать без видимого источника и права использовать в публичном тексте. |
| Отключение SCR/AdBlue | Не предлагать и не описывать. |
| Изготовление РВД | Не писать, пока подтвержден только ремонт и замена РВД. |
| Страницы "услуга x марка" | Не создавать без отдельного решения и самостоятельного контента. |
