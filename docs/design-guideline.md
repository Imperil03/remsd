# REMSD Design Guideline

## Направление

Индустриальный B2B-сервис: ремонтная база, тяжёлая техника, официальные сервисные компетенции, быстрый контакт с мастером. Визуальный язык должен выглядеть технически, спокойно и надёжно.

## Цвета

- Основной фон: `#0b1220`, `#111827`
- Поверхности: `#ffffff`, `#f4f7fb`, `#e8edf5`
- Текст: `#102033`, `#5d6878`, `#ffffff`
- Фирменный синий: `#2f58a8`
- Акцент/CTA: `#f5a21a`
- Линии и разделители: `rgba(16, 32, 51, 0.12)`

## Типографика

- Заголовки: плотный гротеск с техническим характером.
- Основной текст: системный sans-serif для скорости и читаемости.
- Цифры, статусы, бейджи: короткие строки, крупный вес, без декоративности.

## Компоненты

- Радиус: 8px для кнопок, форм и карточек.
- Кнопки: синий для вторичных действий, жёлтый для основной заявки.
- Формы: высокие поля 48-52px, видимый focus, понятные ошибки.
- Карточки: только для повторяющихся сущностей: услуги, техника, сертификаты.

## Контент

- Не использовать фейковые цифры, отзывы и сроки.
- Важные доказательства: сертификаты, модели техники, адрес, телефон, ремонтная база.
- Тон: профессионально, конкретно, без рекламных штампов.
